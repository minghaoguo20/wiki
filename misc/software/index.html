    zotero

    $1 annotation color

    • 黄色和橘色：常规标记，交替使用
    • 灰色：背景知识，非重点内容
    • 绿色(粉色)：重点内容
    • 蓝色：idea
    • 红色：段首单词，本段翻译

    • 紫色：稍后学习

    • Yellow & orange: Regular tags, used interchangeably.

    • Gray: Background knowledge, non-key content.
    • Green(Pink): Key content.
    • Blue: Idea.
    • Red: First word of the paragraph, translated in this paragraph.
    • Purple: To be learned later.

    vs code

    $1 python format

    black

    1. vs code 插件安装 black
    2. 右键 -> Format Document With... -> black
    3. 右键 -> Format Document With... -> Configure Default Formatter -> black

    $2 运行快捷键

    • 快捷键设置
    • command+, -> keyboard shortcut

    • map “Ctrl+R” to “run python files in terminal”

    • debug 运行到当前行

    • option+shift+c
    • 私人订制，c代表cursor
